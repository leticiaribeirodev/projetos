# Landing Page - Empresa Digital 🚀

## 📋 Descrição

Landing page profissional moderna para agência digital "Empresa Digital".

**Recursos:**

- ✅ Design responsivo (mobile-first)
- ✅ Animações AOS + contadores animados
- ✅ SEO otimizado (meta tags, JSON-LD)
- ✅ Formulário de contato (Formspree - configure ID)
- ✅ Performance otimizada
- ✅ Navbar fixa com menu mobile
- ✅ Scroll suave + botão voltar ao topo

##  Como usar

1. Abra `index.html` no browser
2. Configure Formspree: [formspree.io](https://formspree.io) → substitua `SEU_FORM_ID`
3. Hospede em Netlify/Vercel/GitHub Pages

##  Tech Stack

```
HTML5 | CSS3 | JavaScript | AOS | Font Awesome
```
